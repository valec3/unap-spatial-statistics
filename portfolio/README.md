# 🚀 Spatial Stats Portfolio UI

Este es el motor visual del proyecto. Una Single Page Application (SPA) de alto rendimiento diseñada bajo la estética **HUD (Heads-Up Display)** para visualizar el progreso académico y los resultados del análisis espacial.

## 🛠️ Tech Stack

- **Core:** [React 18](https://reactjs.org/) + [TypeScript](https://www.typescriptlang.org/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) + Custom CSS for HUD effects.
- **Animations:** [Framer Motion](https://www.framer.com/motion/) (para transiciones suaves y micro-interacciones).
- **Icons:** [Lucide React](https://lucide.dev/).
- **Engine:** [Bun](https://bun.sh/) (Fastest JS runtime).

## ⚡ Desarrollo

Este proyecto utiliza **Bun** para una gestión de paquetes y ejecución de scripts ultra rápida.

```bash
# Instalar dependencias
bun install

# Levantar servidor de desarrollo
bun run dev

# Construir para producción
bun run build
```

## 🏗️ Estructura del Código (Refactorizado)

- `src/components/hud/`: Componentes tácticos (TaskTiles, ProjectModals, HUD Borders).
- `src/data/`: Centralización de unidades, tareas y metadatos del curso.
- `src/pages/`: Vistas de alto nivel y layouts principales.
- `.atl/`: Metadatos de Spec-Driven Development.

## 🎨 Design Philosophy
Buscamos una interfaz que no solo informe, sino que **impacte**. Cada sección del portfolio está diseñada para sentirse como un "Command Center", reflejando la precisión técnica y el rigor del análisis espacial de datos.
