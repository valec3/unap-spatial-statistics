\section{Descripción del Proyecto}
Este proyecto corresponde a la \textbf{Tarea 03} del curso de Estadística Espacial. El objetivo central es la automatización completa de los procesos de extracción, limpieza y consolidación de datos provenientes de 14 reportes PDF generados por el software de gestión ganadera \textbf{DairyPlan}.

La solución implementada transforma reportes desestructurados en un dataset unificado de alta calidad, permitiendo una toma de decisiones basada en datos para la gestión productiva y reproductiva de planteles lecheros.

\section{Metodología y Tecnologías}
Se utilizó el lenguaje de programación \textbf{R (v4.4.1)} como núcleo del procesamiento. El pipeline se divide en tres etapas fundamentales:

\begin{enumerate}
    \item \textbf{Extracción (OCR/Regex):} Uso de \texttt{pdftools} y expresiones regulares para capturar tablas desde los archivos PDF originales en la carpeta \texttt{/Input/}.
    \item \textbf{Consolidación:} Un script maestro (\texttt{consolidar\_maestro\_v2.R}) une las 14 fuentes individuales mediante operaciones de \texttt{full\_join}, resolviendo inconsistencias y normalizando el \textit{Status Reproductivo}.
    \item \textbf{Visualización:} Desarrollo de un Dashboard interactivo en \textbf{Shiny} utilizando \texttt{bslib} para la interfaz y \texttt{plotly} para gráficos dinámicos.
\end{enumerate}

\section{Indicadores Clave de Rendimiento (KPIs)}
El sistema monitorea métricas críticas para la eficiencia ganadera:
\begin{itemize}
    \item \textbf{DEL (Días en Lactancia):} Promedio de días desde el último parto, indicador de eficiencia de ordeño.
    \item \textbf{Lactancia Promedio:} Número medio de partos por vaca en el plantel.
    \item \textbf{Status Repro:} Clasificación inteligente entre vacas preñadas, inseminadas y vacías.
\end{itemize}
